欢迎来到我的博客

这是使用 Astro + Tailwind CSS 构建的现代化博客。

特性

这个博客具备以下功能：

  • 🌙 深色/浅色主题切换
  • 🔍 全文搜索
  • 📱 响应式设计
  • 📰 RSS 订阅
  • 💬 Giscus 评论系统
  • 🏷️ 标签分类
  • 🔧 SEO 优化

快速开始

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
